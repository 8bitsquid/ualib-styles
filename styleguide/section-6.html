Grid & Gutters 6.0

The grid and gutters provide a framework to build responsive content layouts while maintaining consistent UX.

The Grid 6.1

The grid has not been altered from Bootstrap's grid. Refer to the documentation for Bootstrap's grid system for more details.

Grid Basics 6.1.1

Grids are defined by column nested inside a row. Grids must wrapped in a container element, using either .container or .container-fluid class (Bootstrap containers). NOTE: The main template will automatically wrap page content in a container element. In the rare event that you need to wrap a grid in a container, keep in mind neither container class is nestable (i.e., only one container per grid, even grids with nested rows).

Nested rows don't need their own cotnainer and still have a 12 column grid, relative to the width of their parent column

Gutters 6.2

Gutters define the space between content blocks and other page elements. Always adhere to the gutter definitions in this section.

Content block gutters 6.2.1

The base gutter is 30px. This is based upon the gutter size between columns in the grid system.

For nested content blocks, you may need smaller gutters:

  • Nested gutter sizes should be a divisor of 30px. The size should scale relative to the size of the parent.
    • First level nesting: from 10px to 15px
    • Second level nesting: 5px
