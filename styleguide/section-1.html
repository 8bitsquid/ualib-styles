Colors 1.0

Always try to stick to the Primary Colors. Only use Secondary Colors for slight accents or highlights. Secondary Colors should only be dominant when used in a child/sub theme.

Primary Colors 1.1

Try to use these colors for everything. Only use these colors where deemed appropriate via this styleguide.
